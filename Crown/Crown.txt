自作ライブラリをまとめたものだよ☆



Crown : 自作ライブラリ達のネームスペースだよ☆

Device : 自作ライブラリ達で連携、管理する為のクラスだよ☆ シングルトンだよ☆

	//	利用者向けインターフェース☆
	IRender : 利用者側から描画システムを制御する為のインターフェース用クラスだよ☆

	//	システム系☆
	System : OSなどの環境に依存する部分を抽象化したものだよ☆
		Window	: ウィンドウを抽象化したものだよ☆

	//	描画系☆
	RenderSystem : グラフィックAPIを抽象化したものだよ☆ OSに依存する部分なのでSystemクラスを継承したクラスが生成を行うよ☆
	RenderingPipeline : 具体的な描画の仕方を決定する部分だよ☆

	//	入力系☆
	Input : 入力系を受け取る為のクラスだよ☆　OSに依存する部分なのでSystemクラスを継承したクラスが生成を行うよ☆

	//	その他☆
	Math : 数学系の関数と構造体をまとめたものだよ☆
